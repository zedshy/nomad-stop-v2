# Restaurant Owner Feature Requests

## Requested Features

1. ✅ **Multiple Add-ons with Required/Optional** - Allow multiple add-ons per item, mark as required or optional
2. ✅ **Meal Option with Drink Selection** - Create combo meals where customers can choose a drink
3. ✅ **Category Reordering** - Ability to move/reorder categories
4. ✅ **Print Order** - Print order functionality in admin panel
5. ✅ **Email Notifications for Accept/Decline** - Email notifications when orders are accepted or declined
6. ✅ **Pizza Base and Toppings** - After selecting pizza size, choose base type and multiple toppings
7. ✅ **Customer Notes Per Item** - Allow customers to add notes to each item in their order
