# How to Open Terminal in Cursor

## Quick Methods:

### Method 1: Keyboard Shortcut âš¡ (Easiest)
- **Mac:** Press `Control + ` ` (Control + Backtick)
- **Windows/Linux:** Press `Ctrl + ` ` (Ctrl + Backtick)

The backtick key (`) is usually located:
- Above the `Tab` key
- To the left of the `1` key
- Below the `Esc` key

### Method 2: Menu ğŸ“‹
1. Click on **View** in the top menu
2. Select **Terminal**
3. A terminal panel will open at the bottom

### Method 3: Command Palette ğŸ¯
1. Press `Ctrl + Shift + P` (or `Cmd + Shift + P` on Mac)
2. Type: `Terminal`
3. Select: **Terminal: Create New Terminal**

### Method 4: Bottom Panel ğŸ”½
1. Look at the bottom of Cursor window
2. You'll see tabs like: "Problems", "Output", "Debug Console"
3. Click the **+** icon or look for a **Terminal** tab
4. Click on it to open

## Once Terminal Opens:
You'll see something like:
```bash
nomadadmin@MacBook-Pro Nomad-Stop-NextJS %
```

Then you can run commands like:
```bash
ssh nomadadmin@92.205.231.55
```

## Still Can't Find It?
Try:
1. Press `Ctrl + Shift + J` (might open the panel)
2. Check if Cursor has a "Panel" menu item
3. Restart Cursor if nothing works
